WHAT IS THIS SITE?

This site displays the lunch and dinner menu for Arrillaga Family Dining Commons, but generally that's representative of all stanford dining halls' menus. Unfortunately, I can't predict specials, so you won't find those here.

WHY?

I built this site to learn web development. It's not too seriousâ€”think programming as home-cooking. I realized it would be cool if someone made a prettier version of the Stanford dining hall menu, one that was easier to navigate. So I built it! API

Access the current meal's menu at entree.today's API.

TECH STUFF

Before I built this, I'd never written to a real database or read from one, either. That means this site is pretty janky. You'll notice the load time is a bit slow. That's really frustrating to me because I built this to make viewing the menu more pleasant. To be clear, that's totally my fault: the code that reads the database when you go to entree.today runs in the browser. I now know it should run on a server. I just don't know enough to actually make that happen. so anyway, longer, weirder loading times are here to stay for now. An August 2022 update: entree.today is now served statically, because a python script running in the cloud updates the index.html and redeploys the file to my hosting provider twice a day. Yay!
